# Yorumator Frontend

Next.js 14 App Router arayüzü. Tailwind, React Server Components ve ISR destekli.

## Kurulum
```bash
pnpm install
pnpm dev
```

`.env.example` dosyasını `.env.local` olarak kopyalayın ve API adresini güncelleyin.

## Özellikler
- Auth route group (login/register)
- Ürün kataloğu sayfası
- API helper katmanı
- ReCATPCHA ve güvenlik başlıkları için hazır env alanları
